### 🧠 Project Overview
This project demonstrates a **Plasma Thruster Prototype** built using high-voltage ionization principles.  
It converts electrical energy into thrust through plasma discharge — a concept inspired by advanced propulsion systems.  

⚙️ **Key Components:** Flyback Transformer, High-Voltage Electrodes  
🔬 **Applications:** Experimental propulsion, plasma physics education, and electronics research
